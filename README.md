# Nexa AI

A modern AI-powered chat application built with Vanilla JavaScript, Express.js, and IndexedDB.

Nexa AI provides a ChatGPT-like experience with persistent chat history, Markdown rendering, dark mode support, and local storage using IndexedDB.

---

## ✨ Features

* 🤖 AI-powered conversations
* 💾 Persistent chat history with IndexedDB
* 📝 Markdown response rendering
* 🌙 Dark / Light theme
* 📱 Fully responsive design
* ⚡ Fast and lightweight Vanilla JavaScript architecture
* 🔄 Real-time message streaming effect
* 📂 Conversation management
* 🎨 Modern chat interface

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Parham-Saravani/Nexa-AI.git
```

### Navigate to Project

```bash
cd Nexa-AI
```

### Install Dependencies

```bash
npm install
```

### Start Backend

```bash
npm run server
```

### Start Frontend

```bash
npm run dev
```
