# Nexa AI

A modern AI-powered chat application built with Vanilla JavaScript, Express.js, and IndexedDB.

Nexa AI provides a ChatGPT-like experience with persistent chat history, Markdown rendering, dark mode support, and local storage using IndexedDB.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Parham-Saravani/Nexa-AI.git
```

### Navigate to Project

```bash
cd Nexa-AI
```

### Install Dependencies

```bash
npm install
```

### Start Backend

```bash
npm run server
```

### Start Frontend

```bash
npm run dev
```
